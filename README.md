# Gold Master Egypt

نظام إدارة محلات الذهب والمجوهرات الذكي المصمم خصيصاً للسوق المصري، يدعم العمل بدون اتصال بالإنترنت (Offline-First) وكأيقونة تطبيق مستقل (Web App / PWA) على الهواتف والأجهزة اللوحية والكمبيوتر.

## المميزات الرئيسية
- **إدارة كاملة للمخزون:** تتبع عيار 21، عيار 18، الفروع، والمشغولات والذهب الكسر.
- **تطبيق ويب مستقل (PWA):** إمكانية التثبيت مباشرة كأيقونة على شاشات الهواتف والكمبيوتر من خلال الإعدادات.
- **دعم كامل بدون إنترنت:** تخزين البيانات محلياً مع المزامنة التلقائية.
- **واجهة مستخدم احترافية وفخمة:** تصميم عصري وأنيق مستوحى من بريق الذهب وداعم للغات والاتجاهات العربية بالكامل.

## التشغيل المحلي (Run Locally)

**المتطلبات الأساسية:** تثبيت Node.js على جهازك.

1. **تثبيت الحزم البرمجية:**
   ```bash
   npm install
   ```

2. **تشغيل التطبيق في بيئة التطوير:**
   ```bash
   npm run dev
   ```

3. **بناء النسخة النهائية:**
   ```bash
   npm run build
   ```
